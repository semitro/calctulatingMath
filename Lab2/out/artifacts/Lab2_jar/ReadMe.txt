Подсчёт определённого интеграла методом Симпсона.
(разработано для лабораторной работы номер 2, вычмат)

Использование:
	Запустить lab2.jar из директории, в которой он лежит, убедившись,
	что рядом с ним находится каталог DymaicFunction 
	(и в нём есть подготовленный файл Function.java)

Ключевые особенности:
	Вводимая Вами функция компилируется внутрь метода Java,
	возвращающего double. Просто поисходит подстановка после
	ключевого слова return;
	
	Такая реализация позволяет Вам:
	
	*Использовать тернарный оператор
		(ex:  x > 0 ? x + 2 : -x)
		(если x > 0, вернуть x + 2 , иначе вернуть -x)
	Тернарный оператор можно вкладывать друг в друга
	
	*Использовать функции стандартной библиотеки языка Java
		(cos(x), log(sqrt(x)), и т.д.)
 
Системные требования:
	Для работы программы необходимы установленные JVM, Javafx и JDK 
	(из командной строки должна запускаться программа javac)

Известные проблемы:
	1) Не всегда отображается жди-меня окошко
	2) Если функция не определена на области интегрирования,
	   потребуется перезапуск программы
